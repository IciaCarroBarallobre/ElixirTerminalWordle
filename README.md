# ElixirTerminalWordle
A wordle game where all the words are related to Elixir. It's played in IEX shell.

Git clone & play it!

```Elixir 
> iex.bat -S mix #Windows
> iex -S mix #Linux
iex > WordleTerminalGame.start()
```
